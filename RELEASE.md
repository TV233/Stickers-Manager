# Release Notes

## v1.0.0

### 新特性
- 支持图片分组管理
- 支持一键复制到剪贴板
- 支持 GIF 动画预览
- 支持多种图片格式 (JPG, PNG, GIF, WebP)
- 响应式界面设计

### 系统要求
- Linux x86_64
- Python 3.12+

### 安装说明
1. 下载 StickersManager.AppImage
2. 添加执行权限：
   ```bash
   chmod +x StickersManager.AppImage
   ```
3. 直接运行：
   ```bash
   ./StickersManager.AppImage
   ```
